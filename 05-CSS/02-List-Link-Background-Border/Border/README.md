# Borders


1. Individual Border Properties

- border-top, border-right, border-bottom, border-left.

Contoh:

```CSS
div {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
}
```

2. Shorthand border

- Menyatukan width, style, dan color.

```CSS
div {
  border: 1px solid black;
}
```

3. border-radius

- Membuat sudut melengkung.

```CSS
img {
  border-radius: 10px;
}
```

4. border-style

- Nilai: solid, dashed, dotted, double.

```CSS
div {
  border-style: dashed;
}
```
