#  Links (Tautan)


## 	Pseudo-class untuk link:

• :link → link normal.

• :visited	→ link yang sudah dikunjungi.

• :hover → saat kursor berada di atas link.

• :active → saat link diklik.

• 	Urutan terbaik: LVHA (Link, Visited, Hover, Active).

Contoh:

```css
a:link { color: blue; }
a:visited { color: purple; }
a:hover { color: red; }
a:active { color: orange; }
a:focus { outline: 2px solid orange; }
```
