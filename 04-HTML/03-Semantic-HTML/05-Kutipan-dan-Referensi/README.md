# 5. Kutipan dan Referensi
---

- Elemen Kutipan Blok **(blockquote)** : 

digunakan untuk mewakili bagian yang dikutip dari sumber lain. Elemen ini memiliki citeatribut. Nilai citeatribut tersebut adalah URL sumbernya.

```html
<blockquote cite="https://example.com">
  "Belajar adalah investasi terbaik."
</blockquote>
```

- Elemen kutipan **(cite)** : 

digunakan untuk menunjukkan sumber karya yang dirujuk secara visual. Menandai judul referensi

```html
<p>Artikel <cite>Belajar HTML</cite> sangat membantu.</p>
```

- **q** Elemen Kutipan Sebaris: 

digunakan untuk menampilkan kutipan sebaris pendek

```html
<p>Dia berkata <q>Belajar itu menyenangkan</q>.</p>
```

```html
<q cite="https://example.com">
  "Belajar adalah investasi terbaik."
</q>
```
