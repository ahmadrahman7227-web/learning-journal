# 2. Elemen Struktur Utama
---

- **Header**: Bagian atas dokumen/section.

```html
<header>
  <h1>Website Saya</h1>
</header>
```

- **Main**: Konten utama halaman.

```html
<main>
  <p>Ini isi utama halaman.</p>
</main>
```

- **Section**: Membagi konten ke bagian lebih kecil.

```html
<section>
  <h2>Artikel</h2>
  <p>Isi artikel...</p>
</section>
```

- **Nav**: Bagian navigasi.

```html
<nav>
  <a href="#home">Home</a>
  <a href="#about">Tentang</a>
</nav>
```

- **Figure & Figcaption**: Untuk gambar/diagram dengan keterangan.

```html
<figure>
  <img src="diagram.png" alt="Diagram">
  <figcaption>Diagram proses kerja</figcaption>
</figure>
```

- **footer Elemen** : Elemen ini ditempatkan di bagian bawah dokumen HTML dan biasanya berisi informasi hak cipta dan tautan halaman penting lainnya.

```html
<main>Konten utama</main>
<footer>&copy; 2026 Ahmad</footer>
```
