# 4. Daftar Istilah dan Penjelasan
---

- Elemen Daftar Deskripsi **(dl)** :

digunakan untuk merepresentasikan daftar pengelompokan istilah-deskripsi

- Deskripsi Istilah **(dt)** elemen : 

digunakan untuk mewakili istilah yang sedang didefinisikan.

- Elemen Detail Deskripsi **(dd)** : 

digunakan untuk mewakili deskripsi istilah.

```html
<dl>
  <dt>HTML</dt>
  <dd>Bahasa markup untuk web.</dd>
  <dt>CSS</dt>
  <dd>Bahasa styling untuk web.</dd>
</dl>
```
