# 3. Elemen Teks
---

- **Emphasis (em)**: Penekanan teks.

```html
<p>Belajar itu <em>penting</em>.</p>
```

- **Idiomatic (i)**: Menandai istilah asing/teknis.

```html
<p>Kata <i lang="fr">bonjour</i> berarti halo dalam bahasa Prancis.</p>
```

- **Strong (strong)**: Penekanan kuat.

```html
<strong>Perhatian!</strong> Jangan lewatkan deadline.
```

- **Bring Attention (b)**: Menyorot teks tanpa makna penting.

```html
<p>Produk <b>SuperPhone</b> baru saja rilis.</p>
```
