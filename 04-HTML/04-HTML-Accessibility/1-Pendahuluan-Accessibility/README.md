# 1. Introduction to Accessibility
---

• 	Tujuan: Membuat konten web dapat diakses oleh semua orang, termasuk penyandang disabilitas.

## WCAG (Web Content Accessibility Guidelines): Standar internasional dengan prinsip POUR:

• 	Perceivable → Informasi harus bisa dirasakan (misalnya teks alternatif untuk gambar).
• 	Operable → Navigasi bisa dilakukan (misalnya lewat keyboard).
• 	Understandable → Konten mudah dipahami.
• 	Robust → Konten bisa diakses oleh berbagai teknologi bantu.
