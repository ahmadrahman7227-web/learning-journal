# 2. Assistive Technology
---

• 	Screen readers: Membacakan isi layar untuk pengguna tunanetra.
• 	Large text / braille keyboards: Membantu pengguna dengan gangguan penglihatan.
• 	Screen magnifiers: Membesarkan tampilan layar.
• 	Alternative pointing devices: Joystick, trackball, touchpad untuk pengguna dengan keterbatasan motorik.
• 	Voice recognition: Mengontrol komputer dengan suara.
