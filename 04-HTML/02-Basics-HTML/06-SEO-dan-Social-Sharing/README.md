# 6. SEO dan Social Sharing

## **SEO**: Search Engine Optimization is a adalah praktik yang mengoptimalkan halaman web agar lebih mudah terlihat dan mendapat peringkat lebih tinggi di mesin pencari.

## **Meta (description) Element**: Ini digunakan untuk memberikan deskripsi singkat untuk halaman web dan berdampak pada SEO.

```html
<meta name="description" content="Tips berkebun di lahan sempit">
```

## **Open Graph Tags (untuk media sosial)**:

- Protokol Open Graph memungkinkan Anda mengontrol bagaimana konten situs web Anda muncul di berbagai platform media sosial, seperti Facebook, LinkedIn, dan banyak lagi.

- Dengan mengatur properti Open Graph ini, Anda dapat menarik pengguna untuk mengklik dan berinteraksi dengan konten Anda. Anda dapat mengatur properti ini melalui kumpulan elemen meta di dalam bagian **head HTML** Anda.

```html
<meta property="og:title" content="Belajar HTML">
<meta property="og:type" content="website">
<meta property="og:image" content="gambar.png">
<meta property="og:url" content="https://example.com">
```
- **og:title Properti** : Ini digunakan untuk mengatur judul yang ditampilkan untuk postingan media sosial.

- **og:type Properti** : typeProperti digunakan untuk mewakili jenis konten yang dibagikan di media sosial. Contoh konten ini meliputi artikel, situs web, video, atau musik.

- **og:image Properti** : Ini digunakan untuk mengatur gambar yang ditampilkan untuk unggahan media sosial.

- **og:url Properti** : Ini digunakan untuk mengatur URL yang akan diklik pengguna untuk postingan media sosial.
