# 1. Dasar-Dasar HTML (Hyper Text Markup Language)

- Peran HTML: Menentukan struktur dan konten halaman web.

- Elemen HTML : Elemen adalah blok penyusun dokumen HTML. Elemen mewakili judul, paragraf, tautan, gambar, dan banyak lagi. Sebagian besar elemen HTML terdiri dari tag pembuka ( <elementName>) dan tag penutup ( </elementName>).

## Berikut sintaks dasarnya:

```html
<elementName>Content goes here</elementName>
```

## Elemen HTML: Bangunan dasar dokumen HTML, biasanya terdiri dari opening tag dan closing tag.

```html
<p>Ini adalah paragraf</p>
```

## Void Elements: Elemen tanpa konten, hanya memiliki start tag.

```html
<img src="gambar.png" alt="Deskripsi gambar">
<meta charset="UTF-8">
```


## Atribut: Memberikan informasi tambahan pada elemen.

```html
<element attribute="value"></element>

<input type="text" placeholder="Masukkan nama">
```

## Boolean attributes: Atribut boolean adalah atribut yang dapat ada atau tidak ada dalam tag HTML. Jika ada, nilainya adalah true, jika tidak, nilainya adalah false. Contoh atribut boolean meliputi 
**disabled, readonly, dan required.**

## Komentar: Catatan dalam kode, tidak ditampilkan di browser.

```html
<!--This is an HTML comment.-->
```
