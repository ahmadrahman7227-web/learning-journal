# 4. Karakter Khusus dan Linking

## HTML Entities: Menampilkan karakter khusus.

```html
<p>Gunakan &lt;img&gt; untuk gambar</p>
```

## Link Element: Menghubungkan CSS eksternal.

```html
<link rel="stylesheet" href="style.css">
```

## Script Element: Menyisipkan JavaScript.

```html
<script src="app.js"></script>
```
