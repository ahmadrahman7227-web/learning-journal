# 9. Target Attribute pada Link

## **target Atribut** : Atribut ini memberi tahu browser ke mana harus membuka URL untuk elemen jangkar.

- **_self**: Buka di tab saat ini.
- **_blank**: Buka di tab baru.
- **_parent**: Buka di parent frame.
- **_top**: Buka di jendela penuh.

```html
<a href="https://freecodecamp.org" target="_blank">Kunjungi</a
```
