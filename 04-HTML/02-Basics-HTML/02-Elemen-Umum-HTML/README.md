# 2. Elemen HTML Umum

## Heading (h1–h6): Judul dengan tingkat kepentingan berbeda.

```html
<h1>Judul Utama</h1>
<h2>Sub Judul</h2>
```

## Paragraf (p): Menampilkan teks paragraf.

```html
<p>Ini adalah paragraf.</p>
```

## Gambar (img): Menyisipkan gambar dengan  dan .

```html
<img src="foto.jpg" alt="Foto profil">
```

## Body: Konten utama dokumen HTML.

```html
<body>Konten halaman</body>
```

## Section & Div:
• section → untuk membagi konten secara semantik.
• duv → wadah umum tanpa makna semantik.

```html
<section>
  <h2>Artikel</h2>
  <p>Isi artikel...</p>
</section>
<div class="container"></div>
```

## Anchor (a): Membuat tautan.

```html
<a href="https://example.com">Kunjungi Situs</a>
```

## List (ul, ol, li): Elemen Daftar Tidak Berurutan (ul) dan Berurutan (ol) : Untuk membuat daftar berpoin, Anda harus menggunakan ul elemen dengan satu atau lebih li elemen yang bersarang di dalamnya seperti ini:

```html
<ul>
  <li>Item 1</li>
 <li>Item 2</li>
</ul>
<ol>
  <li>Langkah 1</li>
  <li>Langkah 2</li>```
</ol>
```

## Emphasis (em) & Strong (strong): Memberi penekanan teks.

```html
<em>Teks miring</em>
<strong>Teks penting</strong>
```

## Figure & Figcaption: Gambar dengan keterangan.

```html
<figure>
  <img src="diagram.png" alt="Diagram">
  <figcaption>Diagram alur kerja</figcaption>
</figure>
```

## Main & Footer:

### main Elemen : Elemen ini digunakan untuk menampilkan konten utama sebuah halaman web.

### footer Elemen : Elemen ini ditempatkan di bagian bawah dokumen HTML dan biasanya berisi informasi hak cipta dan tautan halaman penting lainnya.

```html
<main>Konten utama</main>
<footer>&copy; 2026 Ahmad</footer>
```
