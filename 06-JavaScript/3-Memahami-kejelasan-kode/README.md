## Peran Titik Koma **(;)** dalam JavaScript dan Pemrograman

- Dalam JavaScript dan banyak bahasa pemrograman lain titik koma **(;)**membantu memisahkan pernyataan dan meningkatkan keterbacaan kode.

- Dalam JavaScript, titik koma **(;)** digunakan untuk menunjukan akhir sebuah pernyataan.

Sama seperti titik (.) yang menandai akhir kalimat dalam bahasa inggris, titik koma **(;)** menandakan akhir baris kode yang dapat dieksekusi. Hal ini memungkinkan mesin JAvaScript untuk membedakan antara perintah-perintah yang terpisah.

```js
let a = 1; // statement ends here
```

## Apa itu komentar pada JavaScript dan kapan harus menggunakannya ?

- Komentar dalam pemrograman digunakan untuk memberikan konteks tambahan pada kede atau meninggalkan catatan untuk diri sendiri dan orang lain.

- Komentar adalah baris atau blok teks yang diabaikan oleh mesin JavaScript saat kode anda dieksekusi.

-Komentar adalah alat yang ampuh untuk membantu menjelaskan kode yang membingungkan, terlalu rumit, atau ditulis dengan buruk.

### Jenis Komentar

- Komentar dengan dua garis miring **(//)**

```js
// I am single line comment
```
Jenis komentar ini cocok untuk penjelasan atau klarifikasi singkat.

- Komentar multi-baris

```js
/*
I am a multiline coment.
This is helpful for longer explanation.
*/
```
Komentar multi-baris berguna ketika anda perlu menulis deskripsi, penjelasan, atau catatan yang lebih panjang dalam kode.
