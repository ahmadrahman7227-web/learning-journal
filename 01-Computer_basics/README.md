# Computer Basics
Menjelaskan dasar dasar komputer yang harus diketahui.

## Perbedaan Perangkat Lunak Sumber Terbuka vs. Sumber Tertutup

### Sumber Terbuka (Open Source)
- Kode program dapat diakses, dipelajari, dimodifikasi, dan didistribusikan oleh siapa saja.
- Contoh: Linux, Apache, LibreOffice.
- Kelebihan: transparan, fleksibel, komunitas besar.
- Kekurangan: dukungan resmi terbatas, kualitas tergantung komunitas.
### Sumber Tertutup (Closed Source)
- Kode program tidak tersedia untuk umum, hanya pengembang resmi yang bisa mengubahnya.
- Contoh: Windows, Microsoft Office, Adobe Photoshop.
- Kelebihan: biasanya ada dukungan resmi, lebih stabil untuk bisnis.
- Kekurangan: tidak bisa dimodifikasi bebas, sering berbayar.

## The operating system's job
**operating system (OS)** manages all of the software and hardware on the computer. 

Sebagian besar waktu, ada beberapa program komputer yang berbeda berjalan secara bersamaan, dan semuanya perlu mengakses **central processing unit (CPU)** , **memori** , dan **penyimpanan komputer Anda (storage)**. Sistem operasi mengkoordinasikan semua ini untuk memastikan setiap program mendapatkan apa yang dibutuhkannya.
Tiga sistem operasi yang paling umum untuk komputer pribadi adalah Microsoft Windows , macOS , dan Linux .

## Hardware (Perangkat Keras)
adalah bagian apa pun dari komputer Anda yang memiliki struktur fisik , seperti keyboard atau mouse. Ini juga mencakup semua bagian internal komputer.

## Software (Perangkat Lunak)
Perangkat lunak adalah serangkaian instruksi yang memberi tahu perangkat keras apa yang harus dilakukan dan bagaimana melakukannya . Contoh perangkat lunak meliputi peramban web, permainan, dan pengolah kata.
