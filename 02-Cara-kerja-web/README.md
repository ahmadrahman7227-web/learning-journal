# How Does the Web Work?

## Server Web

Server web adalah komputer atau perangkat lunak yang menyimpan dan menyajikan halaman web kepada pengguna melalui internet. Contoh perangkat lunak server web: Apache, Nginx, Microsoft IIS.

## Jaringan

Jaringan adalah kumpulan perangkat (komputer, ponsel, printer, dll.) yang saling terhubung untuk bertukar data dan sumber daya. Bisa berupa jaringan kecil (LAN) atau besar (WAN).

## Internet

Internet adalah jaringan global yang menghubungkan jutaan perangkat di seluruh dunia menggunakan protokol standar (TCP/IP). Internet memungkinkan komunikasi, berbagi data, dan akses ke berbagai layanan.

## Alamat IP

Alamat IP adalah identitas numerik unik yang diberikan pada setiap perangkat di jaringan agar bisa saling berkomunikasi.
- IPv4: 192.168.1.1
- IPv6: 2001:db8::1

## Router

Router adalah perangkat yang menghubungkan beberapa jaringan dan mengarahkan paket data ke tujuan yang tepat. Misalnya, router rumah menghubungkan jaringan lokal (LAN) ke internet.

## ISP (Internet Service Provider)

ISP adalah penyedia layanan internet yang memberi akses ke jaringan internet. Contoh: Telkomsel, Indihome, Biznet.

## Paket dan Transfer Data

Paket adalah potongan kecil data yang dikirim melalui jaringan.
- Data dipecah menjadi paket-paket.
- Setiap paket berisi header (alamat asal, tujuan, urutan) dan payload (isi data).
- Paket dikirim melalui berbagai jalur dan disusun kembali di perangkat penerima.

## Klien

Klien adalah perangkat atau aplikasi yang meminta layanan dari server. Contoh: browser yang meminta halaman web dari server.

## Server

Server adalah perangkat atau aplikasi yang menyediakan layanan atau data kepada klien. Contoh: server email, server web.

## Halaman Web

Halaman web adalah dokumen digital berbasis HTML yang dapat berisi teks, gambar, video, dan link. Contoh: artikel Wikipedia.

## Peramban Web (Web Browser)

Peramban web adalah aplikasi yang digunakan untuk membuka dan menampilkan halaman web. Contoh: Chrome, Firefox, Edge.

## Mesin Pencari (Search Engine)

Mesin pencari adalah layanan yang membantu menemukan halaman web berdasarkan kata kunci. Contoh: Google, Bing, DuckDuckGo.

## Permintaan DNS

Permintaan DNS adalah proses ketika komputer meminta server DNS untuk menerjemahkan nama domain (misalnya google.com) menjadi alamat IP yang bisa dipahami komputer.

## Browser yang Saya Gunakan

Chrome, Firefox, atau Edge.

## Proses Pencarian di Google.com

- Ketik alamat: Kamu mengetik google.com di browser.
- Permintaan DNS: Browser meminta DNS untuk menerjemahkan google.com menjadi alamat IP server Google.
- Koneksi ke server: Browser menghubungi server web Google menggunakan alamat IP tersebut.
- Server merespons: Server Google mengirimkan halaman web (berisi form pencarian).
- Browser menampilkan halaman: Browser menampilkan halaman Google kepada kamu.
- Ketik kata kunci: Kamu memasukkan kata kunci pencarian.
- Permintaan dikirim: Browser mengirimkan permintaan ke server Google.
- Google memproses: Server Google mencari hasil yang relevan.
- Hasil dikirim: Server Google mengirimkan halaman hasil pencarian.
- Browser menampilkan hasil: Kamu melihat daftar halaman web sesuai kata kunci.
